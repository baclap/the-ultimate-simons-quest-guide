## ROM Files

ROM files are not included in this repository. Development and regeneration of
ROM-derived assets require a local copy at `roms/cv2.nes`.

## ROM-Derived Data

The committed guide runtime assets and supporting data include information
derived from the *Castlevania II: Simon's Quest* ROM, such as tile data,
palettes, layout bytes, actor metadata, text, item icons, and music playback
events. These assets are included to make the guide work as a static,
ROM-faithful reference experience.

## Manuals And Published Sources

Some item text, naming policy, and flavor copy references the original
instruction manual or Nintendo Power material. Those sources are used for
historical reference, naming, and guide context.

## [`cv2r`](https://github.com/tonylukasavage/cv2r) Provenance

Earlier project stages used metadata from [`tonylukasavage/cv2r`](https://github.com/tonylukasavage/cv2r), which is
licensed ISC. The production repository no longer vendors that project. The
needed metadata has been extracted into `data/vendor/cv2r/*.json`, with
provenance recorded in those files.
