# Exterior World Composition Demo

This demo summarizes the second milestone in the two-step transition plan.

It uses `out/exterior-world-composition/world-composition.json`, copied into
`assets/exterior-world-composition/`, plus the generated `world.png`.

The demo shows:

- all 32 exterior topology areas
- all 55 exterior topology nodes
- all 87 topology constraints
- zero hand-placed coordinates
- routine-supported, connector-only, unresolved, conflict, and solver-derived
  placement labels

This is a world composition draft, not the final pixel-perfect whole-game map.
Its value is that the full exterior graph is assembled from ROM-derived data and
deterministic constraints with every remaining gap called out explicitly.
