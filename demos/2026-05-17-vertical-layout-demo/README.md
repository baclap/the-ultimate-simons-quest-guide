# Vertical Layout Demo

This demo shows the first atlas milestone after decoding layout headers as
two-dimensional grids. It compares the previous single-section view against the
new full-grid layout output for locations with vertical map structure.

Run from this directory with any static web server, for example:

```sh
python3 -m http.server 8099 --bind 0.0.0.0
```

The PNGs are generated from ROM layout, tile, CHR, and palette data. They are
not emulator screenshots.
