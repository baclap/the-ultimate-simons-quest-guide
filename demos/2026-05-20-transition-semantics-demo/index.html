Topology

Every edge carries a transition class

The classes come from ROM marker bytes where possible. The tornado case is deliberately marked as a candidate because the endpoint is ROM-derived, while the transport interpretation comes from reverse-engineered metadata and gameplay behavior.

Special Case

Deborah Cliff is no longer treated as ordinary adjacency

The route still connects to Bodley Mansion, but the composition records that exact transport coordinates are unresolved. This removes the old generic overlap shove and replaces it with a named connector break.
