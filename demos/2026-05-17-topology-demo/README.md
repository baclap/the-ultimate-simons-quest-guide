# Exterior Topology Demo

This demo shows the milestone where the exterior atlas becomes topology-aware.
The images still come from the ROM-native exterior atlas renderer, but their
ordering and connections come from decoded ROM area-transition triples.

Open `index.html` directly, or serve this directory with a local static server.

## Assets

- `assets/exterior-atlas/` is a snapshot of the current rendered exterior atlas.
- `assets/topology/topology-data.js` is generated by
  `npm run render:topology:exterior`.

## Commands

```sh
npm run render:atlas:exterior
npm run render:topology:exterior
```
