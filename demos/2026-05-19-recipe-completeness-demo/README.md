# Recipe Completeness Demo

This demo shows the milestone where the render-recipe atlas has no deferred or
diagnostic family left.

It uses generated ROM-derived assets from `out/render-recipe-atlas/` plus one
older Castlevania render from the prior demo as a before image. The before image
documents the old objset `5` CHR assumption (`06/07`); the current render is
derived from the newly audited final-area recipe (`0B/0C`, palette transfer
`$57 -> 4:$A150`).
