# Recipe Resolver Atlas Demo

This demo presents the first atlas generated by `npm run render:recipe-atlas`.
It uses `out/render-recipe-audit/audit.json` as recipe evidence, then renders
99 ROM-native atlas entries:

- 11 validated by exact save-state probes
- 80 projected from validated recipe families
- 8 diagnostic renders for families still awaiting representative states

The copied assets under `assets/recipe-atlas/` are generated outputs from the
local ROM. They are not emulator screenshots.

Open `index.html` directly or serve this directory with a local static server.
