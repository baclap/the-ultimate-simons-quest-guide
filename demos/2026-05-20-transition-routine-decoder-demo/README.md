# Transition Routine Decoder Demo

This demo summarizes the first milestone in the two-step transition plan.

It uses `out/transition-routine/decoder.json`, copied into
`assets/transition-routine/decoder.json` and `decoder-data.js`.

The demo shows:

- 6 scripted probes and 10 completed transitions
- observed `$70-$73` write PCs
- nearby ROM byte windows around those PCs
- per-transition routine bytes, topology edges, Simon placement, and camera state
- promoted, diagnostic, and unresolved role hypotheses

This demo does not render a new map. It creates the evidence model the next
world composition milestone consumes.
