# Overworld Section Label Audit: Three Sources

This report compares the current guide overworld labels against three map
sources:

- Nintendo Power *Castlevania II: Simon's Quest* Transylvania map:
  <https://www.castlevaniacrypt.com/wp-content/img/cv2/np/13.jpg>
- NES Game Atlas map, "Terror-tory of Transylvania":
  <https://www.castlevaniacrypt.com/wp-content/img/cv2/maps/nesmap-2.png>
- User-supplied fan map image attached in chat on 2026-07-01.

This is a review artifact, not a policy change. Nintendo Power and NES Atlas are
published map sources. The fan map is treated as a community route/naming lead:
useful for conflict discovery, but not final authority for guide facts.

## Full Segment Comparison

Status meanings:

- `stable`: current guide label agrees with the source maps at the useful
  player-facing level.
- `broad`: source maps name a broad region while the guide splits ROM/manifest
  sub-sections.
- `conflict`: at least one source clearly gives a different player-facing name.
- `concern`: one of the current labels singled out for policy review.
- `guide-authored`: current label describes guide presentation rather than a
  printed map label.

## Resolved Naming Decisions

These labels were changed after comparing the current guide against Nintendo
Power, NES Atlas, and the fan map.

| Segment id | Current guide label | Decision |
| --- | --- | --- |
| `debious-woods` | `Debious Woods` | Move the Debious label to the section that best fits the source-map region. |
| `uta-lower-road-2` | `Uta Lower Road 2` | Follow NES Atlas `Uta Lower Road-2`, converted to guide numbering style. |
| `uta-lower-road-1` | `Uta Lower Road 1` | Follow NES Atlas `Uta Lower Road`, converted to guide numbering style. |
| `uta-road` | `Uta Road` | Use a short guide-authored connector label rather than inventing `Uta Lower Road 3`. |
| `west-bridge` | `West Bridge` | Follow NES Atlas and the fan map. |
| `east-bridge` | `East Bridge` | Follow NES Atlas and the fan map's nearby bridge naming. |
| `denis-marsh` | `Denis Marsh` | Follow NES Atlas despite remaining fan-map conflict with `Vlad Mountain`. |
| `lower-road` | `Lower Road` | Follow NES Atlas; the fan map uses `Lake Path`. |

## Additional Conflicts To Track

These are not in the active five, but they are real source conflicts:

- `Lauber Mansion` versus `Rover Mansion`: Nintendo Power supports Lauber;
  NES Atlas and the fan map support Rover.
- `Vrad` versus `Vlad`: Nintendo Power and current guide use Vrad for Graveyard
  and Mountain; the fan map uses Vlad.
- Town spellings on the fan map differ from both published maps and current
  guide labels: `Andole`, `Aldora`, and `Donya`.
- `Storigoi Graveyard` becomes `Storigoy Cemetery` on the fan map.
- `Dabi's Path` / `Dabis Path` remains a punctuation/spelling policy question,
  but all sources point to the same route.

## Policy Implications

The fan map helped resolve two source conflicts:

- It agrees with NES Atlas on `West Bridge`, so the guide now uses `West Bridge`
  instead of `Castlevania Bridge`.
- The guide now resolves the former `Yuba Lake Path` concern as `Lower Road`,
  following NES Atlas over the fan map's `Lake Path`.

It complicates two others:

- For the former `Bordia Mountains`, the fan map uses `Vlad Mountain`; the guide
  now follows NES Atlas with `Denis Marsh`.
- For the former `Debious Woods - Part 1`, the fan map suggests the Wicked Ditch
  side while NES Atlas does not provide a third Uta Lower Road label; the guide
  uses the connector label `Uta Road`.
